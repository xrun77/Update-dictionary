Замена словаря в файле omni.ja браузера Firefox / Replacing the dictionary in the omni.ja file of the Firefox browser
1) Запустите файл update_dict.bat
2) Выберите вариант, какая у Вас локализация Firefox: английская или русская.
3) Выберите папку, в которой находится файл omni.ja
4) Дождитесь завершения работы программы. Если все нормально, будет выведено окно с количеством замен. Если нет - смотрите файл %TEMP%\ERROR.log
5) Переименуйте файл omni_mod.ja --> omni.ja и скопируйте с заменой исходного файла.

P.S. Можно автоматизировать весь процесс, если локазизация Firefox и местоположение файла omni.ja постоянные, посмотрите комментарии в файлах update_dict.bat и omni.ja_update.ini.
